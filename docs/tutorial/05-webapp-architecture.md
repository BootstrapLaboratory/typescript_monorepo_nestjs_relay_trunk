# Webapp Architecture

This chapter will explain the frontend design: React, Vite, Relay, TanStack
Router, feature folders, shared browser transport modules, `src/ui`,
vanilla-extract, and Storybook.

## Chapter Intent

Cover why route adapters stay thin, why Relay and realtime transport live in
shared modules, why feature folders own page composition, and how the webapp
build turns deployment-time endpoint values into browser code.
