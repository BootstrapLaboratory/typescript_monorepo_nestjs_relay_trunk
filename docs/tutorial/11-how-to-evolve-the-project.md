# How To Evolve The Project

This chapter will be a design playbook for future changes: adding a GraphQL
field, adding a server module, adding a webapp feature, adding a deploy target,
adding a provider function, and adding a scenario step.

## Chapter Intent

Cover where new behavior should live and which contracts need to be updated
when a change crosses boundaries between server, webapp, schema, deployment,
and pre-deployment setup.
