# CI Validation And Local Workflows

This chapter will explain pull request validation, the main release workflow,
forced target workflows, local development commands, schema generation, Relay
generation, migrations, and validation checks.

Cover how local developer workflows line up with CI workflows, why validation
belongs in project scripts and Rush Delivery metadata, and what checks should
run before deployment.
