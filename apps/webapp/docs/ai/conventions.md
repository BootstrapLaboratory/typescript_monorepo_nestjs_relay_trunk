# Webapp AI Conventions

- Read `apps/webapp/docs/ai/architecture.md` before changing webapp code,
  Cloudflare Pages deployment behavior, Relay operations, or webapp Rush
  Delivery metadata.
- If webapp architecture changes, update `apps/webapp/docs/ai/architecture.md`
  in the same task.
- Keep webapp architecture documentation scoped to the frontend boundary. Do not
  restate repository-level architecture there.
