# Server AI Conventions

- Read `apps/server/docs/ai/architecture.md` before changing server code,
  Cloud Run deployment behavior, backend GraphQL schema generation, database
  migrations, or server Rush Delivery metadata.
- If server architecture changes, update `apps/server/docs/ai/architecture.md`
  in the same task.
- Keep server architecture documentation scoped to the backend boundary. Do not
  restate repository-level architecture there.
