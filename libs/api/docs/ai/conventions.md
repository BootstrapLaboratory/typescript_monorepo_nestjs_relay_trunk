# API Contract AI Conventions

- Read `libs/api/docs/ai/architecture.md` before changing `libs/api` or the
  GraphQL schema contract between server and webapp.
- If API contract architecture changes, update
  `libs/api/docs/ai/architecture.md` in the same task.
- Keep API contract architecture documentation scoped to the shared schema
  contract. Do not restate server or webapp architecture there.
